## API Documentation

### Gulf Standards Lookup API

#### Endpoint: `/api/standards/gso/{code}`

**Description:** Retrieve GSO standard information.

**Parameters:**
- `code` (string): GSO standard identifier
- `country` (optional string): GCC country code (SA, UAE, QA, KW, BH, OM)

**Response:**
```json
{
  "code": "GSO 399",
  "title": "Steel Reinforcement Bars for Concrete",
  "version": "2017",
  "applicable_countries": ["SA", "UAE", "QA", "KW", "BH", "OM"],
  "parameters": {
    "steel_grades": ["300", "400", "500"],
    "yield_strength_range": "300-500 MPa",
    "tensile_strength_min": "480 MPa"
  },
  "national_amendments": {
    "SA": "SBC 302-14",
    "UAE": "Dubai Municipality Amendment",
    "QA": "QCS 2014 Edition"
  }
}
```

{

## Integration Guide for AI Engine

### Contract Generation Integration

**Step 1: Regional Standards Detection**
```python
def detect_gcc_region(contract_text):
    """
    Detect which GCC country/region the contract applies to
    """
    region_keywords = {
        'Saudi': ['Saudi Arabia', 'KSA', 'Riyadh', 'Jeddah'],
        'UAE': ['United Arab Emirates', 'Dubai', 'Abu Dhabi', 'Sharjah'],
        'Qatar': ['Qatar', 'Doha'],
        'Kuwait': ['Kuwait', 'Kuwait City'],
        'Bahrain': ['Bahrain', 'Manama'],
        'Oman': ['Oman', 'Muscat']
    }

    detected_region = None
    for region, keywords in region_keywords.items():
        if any(kw in contract_text for kw in keywords):
            detected_region = region
            break

    return detected_region
```

**Step 2: Standards Mapping**
```python
STANDARDS_MAP = {
    'Saudi': {
        'concrete': 'SBC 302-14 (ACI 318-14)',
        'steel': 'SBC 303-16 (AISC 360-16)',
        'masonry': 'SBC 304-17 (MSJC-13)',
        'seismic': 'SBC 305-21 (ASCE 7-16)',
        'fire': 'Saudi Civil Defense Code'
    },
    'UAE': {
        'concrete': 'Dubai Municipality Concrete Code',
        'fire': 'UAE Fire and Life Safety Code 2018',
        'sustainability': 'Estidama Pearl Rating System'
    },
    'Qatar': {
        'general': 'QCS 2014',
        'sustainability': 'GSAS',
        'fire': 'Qatar Fire Code'
    },
    'Bahrain': {
        'building': 'BMDI Building Code 2022',
        'fire': 'Bahrain Fire Prevention Standards',
        'electrical': 'EWA Distribution Code'
    }
}

def get_regional_standards(region, discipline):
    """
    Get applicable standards based on region and discipline
    """
    return STANDARDS_MAP.get(region, {}).get(discipline)
```

**Step 3: Validation Check**
```python
def validate_gcc_compliance(contract_text, region):
    """
    Validate contract against regional GCC standards
    """
    compliance_issues = []

    # Check for regional standard references
    regional_standards = STANDARDS_MAP.get(region, {})
    for discipline, standard in regional_standards.items():
        if standard not in contract_text:
            compliance_issues.append(f'Missing {discipline} standard reference: {standard}')

    # Check for specific requirements
    if region == 'Saudi':
        if 'SASO' not in contract_text:
            compliance_issues.append('Missing SASO material standards reference')
    elif region == 'UAE':
        if 'UAE Fire' not in contract_text:
            compliance_issues.append('Missing UAE Fire and Life Safety Code reference')

    return compliance_issues
```
